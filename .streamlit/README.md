# ♻️ EcoSort AI: Smart Waste Management

EcoSort AI is a multimodal assistant that uses Google Gemini 2.5 Flash to identify waste and provide sorting guides (SDG 12).

## ✨ Features

- 📷 **Image Recognition**: Identify waste via photos.
- ⌨️ **Smart Search**: Type waste items for instant disposal tips.
- 🛡️ **Secure**: Uses Streamlit Secrets for API management.

## 🚀 Tech Stack

- Python & Streamlit
- Google GenAI SDK (Gemini 2.5 Flash)
